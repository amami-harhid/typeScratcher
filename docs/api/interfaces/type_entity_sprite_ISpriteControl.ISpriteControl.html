    stopOtherScripts

    • stopOtherScripts(): void

      このスプライトの他のスクリプトを停止する

      親スプライトから派生したクローンを含めて、同じスプライトであるとみなす。 クローンからこのメソッドを実行する場合、他のクローン、親スプライトは同じスプライトとはみなさない。 親スプライトからこのメソッドを実行すると、親の他のスクリプトと派生したクローンの全てのスクリプトを停止する。 クローンからこのメソッドを実行すると、当該クローンの他のスクリプトだけを停止する。

      Returns void
