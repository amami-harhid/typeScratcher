# 赤ボタンで停止した後、旗クリックが効かない
赤ボタンで停止した後に旗クリックをすると、動作が始まらない。
赤ボタンで停止するとき、
Proxyで「このスクリプトを止める」フラグオンとしスローを発生させている。フラグオンのままなので再実行ですぐにスローしていた。
フラグをオフに（Proxyを再生成)した。
